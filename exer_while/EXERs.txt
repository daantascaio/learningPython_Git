Exercício 1:
Escreva um programa que imprima os números inteiros de 1 a 10 usando um loop "while".





Exercício 2:
Escreva um programa que solicite ao usuário um número e imprima a contagem regressiva a partir 
desse número até zero.





Exercício 3:
Escreva um programa que calcule a soma de todos os números inteiros de 1 a N, onde N é um número 
inteiro fornecido pelo usuário.





Exercício 4:
Escreva um programa que solicite ao usuário que digite uma senha.
Continue solicitando a senha até que o usuário digite a senha correta.
Quando a senha correta for digitada, imprima uma mensagem de boas-vindas.





Exercício 5:
Escreva um programa que gere a sequência de Fibonacci até o N-ésimo termo, onde N é um número 
inteiro fornecido pelo usuário. A sequência de Fibonacci é uma sequência de números em que cada 
número é a soma dos dois anteriores (começando por 0 e 1).




Exercício 6:
Escreva um programa que imprima os múltiplos de 7 menores que 100 usando um loop "while".




Exercício 7:
Escreva um programa que solicite ao usuário números inteiros. Pare a leitura quando o usuário 
digitar zero e, em seguida, imprima a soma de todos os números digitados.





Exercício 8:
Escreva um programa que simule um jogo de adivinhação. O programa deve gerar um número aleatório 
entre 1 e 100 e, em seguida, solicitar ao usuário para adivinhar o número. O programa deve informar 
se o palpite do usuário é muito alto, muito baixo ou correto. Continue solicitando palpites até que 
o usuário adivinhe corretamente.
